This is the API documentation for the animal adoption application.

Adopters

Create a new adopter

Only admin and staff users can create adopters

Authorizations:
bearerAuth
Request Body schema: application/json
required
name
required
string

Full name of the adopter

email
required
string <email>

Email address of the adopter

phone
required
string

Contact number of the adopter

address
string

Home address of the adopter

preferredSpecies
Array of strings

Preferred animal species for adoption

Responses

Local server

http://localhost:3000/api/v1/adopters

Request samples

  • Payload
Content type
application/json
{
  • "name": "Jane Smith",
  • "email": "jane@example.com",
  • "phone": "+1-204-555-1234",
  • "address": "123 Main St, Winnipeg, MB",
  • "preferredSpecies": [
    • "Dog",
    • "Cat"
    ]
}

  • 400
